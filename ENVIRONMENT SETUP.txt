Um das Projekt am laufen zu kriegen, folgt bitte folgende Schritte:

1) VSCode installieren : https://code.visualstudio.com/download

2) Node.js installieren : https://nodejs.org/en/download

3) VSCode öffnen und links unter VSCode Symbol Extensions finden (STR+SHIFT+X in Windows/ SHIFT+CMD+X in Mac)
    - Prettier installieren
    - TypeScript installieren
    - Vue Official installieren

4) pnpm installieren : einen Terminal mit Adminrechte öffnen und folgende Befehle ausführen:

npm install -g pnpm

5) Drizzle installieren

Auf https://orm.drizzle.team/docs/get-started-postgresql#supabase gehen und Befehle unter Install Dependencies in terminal ausführen

6) supabase installieren

auf dem Terminal pnpm i @nuxtjs/supabase ausführen

in home Verzeichnis nuxt.config.ts Datei finden, da den modules-Array um "@nuxtjs/supabase" ergänzen.

7) Projekt starten

in VSCode neue terminal öffnen, pnpm dev ausführen und link folgen
